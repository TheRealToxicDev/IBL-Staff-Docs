This section will walk you through Setting Up and Configuring the Website and Environment

---

## Environment Structure

```json:title=.env
REACT_APP_CLIENT_ID = '' // Discord Bot ID
REACT_APP_CLIENT_SECRET = '' // Discord Client Secret
REACT_APP_WEBHOOK_URL = '' // Discord Channel Webhook URL
REACT_APP_DISCORD_BOT_TOKEN = '' // Discord Bot Token
REACT_APP_GUILD_ID = '' // Discord Server ID
REACT_APP_JWT_SECRET = '' Some Random Secret Password (for the API)
REACT_APP_SKIP_BAN_CHECK = '' // Skip chekcing the Server if User is Banned (true or false)
```
